
A compilation (AHH!) of small tools and games for the Command Line in ANSI C
by Thomas Gruetzmacher (http://tomaes.32x.de)

    -> NBR_GAME : Mini game
    -> MM_GAME  : A version of 'Mastermind'
    -> PWDGEN   : Password / token generator
    -> MEDT     : Hex-editor / memory viewer / file dumper
    -> PICROSS  : Picross clone*
    -> EST_GAME : Frequency / estimation game
    -> FMERGE   : File appending / filter / conversion tool

See source file for more info on these programs.

latest update: 07/2012

*not finished yet.
